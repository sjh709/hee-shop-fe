## 💻 프로젝트 소개

- 이 프로젝트는 백엔드와 프론트엔드를 구현하여 처음부터 끝까지 서비스를 직접 만들어 보자는 목표를 가지고 제작된 쇼핑몰 웹사이트입니다.
- 장바구니, 상품 주문, 관리자 페이지 등 다양한 쇼핑몰 서비스를 제공합니다.
- 실제 결제 서비스는 구현되지 않았으며 UI만 구현되어 쉽게 테스트를 할 수 있습니다.
