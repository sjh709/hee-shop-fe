## 💻 프로젝트 소개

- 이 프로젝트는 백엔드와 프론트엔드를 구현하여 처음부터 끝까지 서비스를 직접 만들어 보자는 목표를 가지고 제작된 쇼핑몰 웹사이트입니다.
- 장바구니, 상품 주문, 관리자 페이지 등 다양한 쇼핑몰 서비스를 제공합니다.
- 실제 결제 서비스는 구현되지 않았으며, UI만 구현되어 쉽게 테스트를 할 수 있습니다.

## 📌 페이지별 기능

### 💫 랜딩 페이지

- 상품 목록을 볼 수 있습니다.
- 상품을 클릭하면 상품 디테일 페이지를 볼 수 있습니다.
- 카테고리별 상품을 볼 수 있습니다.
- 상품 이름으로 검색이 가능합니다 -> 검색어 입력 후 Enter
- 메뉴를 클릭하면 각각의 페이지로 이동합니다. (로그인, 쇼핑백, 내 주문)
- 로그인이 되었을 경우 로그아웃을 할 수 있습니다.
- 쇼핑백, 내 주문 메뉴는 로그인을 한 경우에만 이용할 수 있습니다.
- 관리자 계정으로 로그인 시 관리자(admin) 페이지로 이동할 수 있습니다.

### 💫 쇼핑백(장바구니) 페이지

- 로그인 된 회원만 이용이 가능한 페이지입니다.

- 주문할 상품의 총금액을 확인할 수 있습니다.
- 결제 계속하기 버튼을 클릭하면 결제 페이지로 이동합니다.

### 💫 관리자 전용 상품 관리 페이지

- 관리자 계정만 접근할 수 있습니다.
- 사이드 메뉴의 로고를 클릭하면 랜딩 페이지로 이동합니다.
- 사이드 메뉴를 클릭하면 각각의 메뉴로 이동합니다. (Product: 상품 관리 페이지, Order: 주문 관리 페이지)
- 상품 관리를 할 수 있는 메뉴입니다.
- 전체 상품 목록을 볼 수 있습니다. (상품 이름, 재고 개수 등 확인 가능)
- 상품 이름으로 검색이 가능합니다.
- 'Add New Item +' 버튼을 클릭하면 상품을 추가할 수 있습니다.
- '-' 버튼을 클릭하면 상품이 삭제됩니다.
- 'Edit' 버튼을 클릭하면 상품을 수정할 수 있습니다.

### 💫 관리자 전용 주문 관리 페이지

- 관리자 계정만 접근할 수 있습니다.
- 사이드 메뉴의 로고를 클릭하면 랜딩 페이지로 이동합니다.
- 사이드 메뉴를 클릭하면 각각의 메뉴로 이동합니다. (Product: 상품 관리 페이지, Order: 주문 관리 페이지)
- 주문 관리를 할 수 있는 메뉴입니다.
- 전체 주문 목록을 볼 수 있습니다. (주문 일자, 주문 상품, 주소, 가격 등 확인 가능)
- 주문 번호로 검색이 가능합니다.
- 주문 목록을 클릭하면 주문 상세 정보를 확인할 수 있습니다.
- 주문 상세 정보 화면에서 주문 상태(Status)를 수정할 수 있습니다.
- 주문 상태 : preparing(상품 준비 중), shipping(배송 중), delivered(배송 완료), refund(반품 신청)
